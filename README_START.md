# 🚀 项目启动指南 (Startup Guide)

如果你重启了电脑，需要重新启动项目，请按照以下步骤操作。

## 方案 A：一键启动 (推荐)
我已经为你写好了一个自动化脚本，只需双击即可。

1. 打开文件夹 `D:\webqimo`。
2. 双击运行 **`start-all.bat`**。
3. 它会自动：
   - 检查并启动 MySQL。
   - 启动 Nginx。
   - 启动 Node.js 后端 (npm start)。
   - 启动 cpolar 内网穿透。

---

## 方案 B：手动启动 (如果脚本运行失败)

如果不使用脚本，你需要按顺序打开 3 个黑窗口（终端）：

### 第 1 步：启动 Node.js 后端
1. 打开 `D:\webqimo` 文件夹。
2. 在地址栏输入 `cmd` 并回车。
3. 输入命令：
   ```bash
   npm start
   ```
   *(保持这个窗口不要关闭)*

### 第 2 步：启动 Nginx
1. 打开 Nginx 目录 (例如 `D:\nginx`)。
2. 双击 `nginx.exe`。
   *(它会闪一下消失，这是正常的，说明在后台运行了)*

### 第 3 步：启动内网穿透 (cpolar)
1. 打开 `Z:\cpolar内网穿透` 文件夹。
2. 在地址栏输入 `cmd` 并回车。
3. 输入命令：
   ```bash
   cpolar http 80
   ```
   *(保持这个窗口不要关闭，复制屏幕上的 http 链接发给别人)*

---

## 常见问题

**Q: 网页打不开？**
- 检查 MySQL 是否启动：按 `Win+R` 输入 `services.msc` 查看 MySQL 服务状态。
- 检查 Nginx 是否启动：在任务管理器里看有没有 `nginx.exe` 进程。

**Q: cpolar 链接变了？**
- 是的，免费版 cpolar 每次重启后链接都会变，需要重新发给别人。
